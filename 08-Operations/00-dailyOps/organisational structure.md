## 1.1 Role Definitions

### SOC Analyst (Tier 1)

Mission:

- Monitor Wazuh dashboard
    
- Investigate alerts
    
- Perform triage
    

Tasks:

- Alert validation
    
- Log review
    
- IOC verification
    
- Ticket creation
    

---

### SOC Analyst (Tier 2)

Mission:

- Incident investigation
    
- Deep log analysis
    
- Threat correlation
    

Tasks:

- Timeline reconstruction
    
- Lateral movement analysis
    
- Memory/process review
    

---

### Detection Engineer

Mission:

- Improve detection coverage
    

Tasks:

- Write Wazuh rules
    
- Tune Suricata rules
    
- Reduce false positives
    
- MITRE mapping
    

---

### Red Team Operator

Mission:

- Simulate adversary behavior
    

Tasks:

- Run attack campaigns
    
- Validate detection gaps
    
- Trigger SOC workflows
